# Kamu Keyboardの組み立て方  

本ガイドはKamu KeyboardのV1を使用して作成されています。  
基本的な組み立て手順は同一ですので、部品を置き換えて組み立てをお願いします。  

# 必要部品  
- PCB   x1  
- ProMicro  x2  
- TRRSジャック  x2  
- タクトスイッチ    x2  
- スイッチソケット（Cherry MX） x10  
- OLED（オプション）    x2  
- アクリルプレート 2mm（トップ）    x2
- アクリルプレート 2mm（ボトム）    x2
- アクリルプレート 2mm（カバー）    x2
- M2 ネジ（皿ねじ以外で3mm以上5mm以下で）   x28
- M2 スペーサー 7mm x14
